# IDEA 编写第一个spark项目

## spark开发流程
1. 官网下载安装Scala and Spark并配置windows环境变量
2. 创建scala项目
3. 导入spark 的jar包
4. 运行第一个demo程序
5. 编译打包
6. 上传到spark集群运行
